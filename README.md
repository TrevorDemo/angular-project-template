Angular Project Template
=======================

A **nice** and **clean** Angualar project template.

### Using

- Angular 1.2.23
- Angular Route
- UI Bootstrap
- Boostrap 3.2.0 (only CSS)

### Also
- Responsive Navbar using UI Boostrap
- Active Link (in Header Controler, `HeaderCtrl`)
